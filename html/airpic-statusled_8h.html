Detailed Description

Status LED manager.

This library uses a common-cathode RGB LED to show system status. It is assumed that the relative brightness of each color is normalized by resistors. For the LED used in testing, a good balance was found with a 200 Ohm resistor connecting to red, a 100 Ohm resistor connecting to green, and a 1000 Ohm resistor connecting to blue.

The LED is attached as follows:

  • Red is attached to RB15 (pin 26 of the PIC24FJ64GA002) through 200 ohm resistor
  • Blue is attached to RB14 (pin 25 of the PIC24FJ64GA002) through 1000 ohm resistor
  • Green is attached to RB13 (pin 24 of the PIC24FJ64GA002) through 100 ohm resistor

By default, after the status LED is initialized by running statusLED_enable(), the LED will show a long, blue blinking pattern.

Use of the status LED should be avoided in a final implementation. The primary purpose of this library is to provide an easy tool to help in debugging by allowing the user to more easily see what the microcontroller is up to. If the status LED isn't being used for debugging or showing vital status updates, it probably shouldn't be used, as it is a little clock cycle-hungry.
